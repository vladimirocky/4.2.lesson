## Шпаргалка
### Чтобы просто получить **html** файл используйте утилиту **pug-cli**
1. Установка **pug-cli**
   - *npm i -g pug-cli*

2. Создание файла **html** из **pug**
   - *pug <from_filename.pug> <to_filename.html>*
   - например: *pug ex1.pug ex1.html*
